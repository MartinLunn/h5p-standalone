# h5p-standalone
Display H5P content with plain old HTML

## Run The Demo (Gulp)

Install dependencies
```
npm install -g gulp
npm install
```

Unzip `.h5p` file into `workspace` directory

```
gulp demo
```

## Run The Demo (Other)

Unzip `.h5p` file into `workspace` directory

Run a simple http server e.g. `python -m SimpleHttpServer`

head to `http://localhost:8080/demo/`
